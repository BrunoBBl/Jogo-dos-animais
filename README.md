# Jogo dos Animais

Plataforma web simples de apostas no estilo tradicional do **Jogo do Bicho**, renomeada como “Jogo dos Animais” para maior discrição e facilidade de hospedagem.

> Atenção: este é um projeto apenas para estudo/demonstração.  
> O Jogo do Bicho é considerado contravenção no Brasil (art. 51 do Decreto-Lei 3.688/41).  
> Use apenas em ambientes privados, de teste ou em locais onde a atividade seja permitida.

## Funcionalidades atuais

- Lista completa dos 25 bichos com dezenas (00–99)
- Registro de apostas (valor, bicho, dezena, centena, milhar)
- Cálculo automático de prêmios (padrão mais comum: 18x no bicho, 100x na centena, 600x no milhar etc.)
- Histórico de resultados e apostas do dia
- Interface limpa e responsiva
- Salva tudo no `localStorage` (não perde ao fechar o navegador)

## Tecnologias

- HTML + CSS + JavaScript puro
- PHP (versão com banco de dados em desenvolvimento)
- Sem frameworks – roda em qualquer hospedagem barata
